# deck

78 tarot cards written as Markdown — each card an **Anchor** (keywords, core meanings) and a **Muse** (artistic direction, impression palette, creator prompt).

Intended as source material for a published grimoire for creators: artists, writers, designers who want to co-create with the cards rather than just look them up.

Start at [`tarot-card-index.md`](tarot-card-index.md).
